Floating menu with cool animation for your iOS app, easy setup and use!

# Installation
#### CocoaPods
```
pod 'HPFloatMenu`
```
#### Manually
Copy the `HPFloatMenu` folder to your project.

## Usage

### Setup
Add `import HPFloatMenu` in your file

## Requirements
Requires Swift 5.0 and iOS 9.0.
